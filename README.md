# GX2GLES
Just a silly project that aims to simulate OpenGL ES 2.0 using GX2.

Right now you can:
- Draw simple stuff on the screen.
- Use shaders
- Work with textures

Basically you can do simple OpenGL stuff.

# Important
I am just a student in software engineering so this project might not be 100% accurate. Please keep that in mind.
